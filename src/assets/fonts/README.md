# Police River Adventurer

Pour utiliser la police "River Adventurer" :

1. Téléchargez la police depuis DaFont : https://www.dafont.com/river-adventurer.font
2. Extrayez le fichier de police (format .ttf, .otf, .woff ou .woff2)
3. Renommez le fichier en `RiverAdventurer-Regular.ttf` (ou .otf, .woff, .woff2 selon le format)
4. Placez le fichier dans ce dossier : `/src/assets/fonts/`

La police sera automatiquement chargée et utilisée pour le style `.logo-titre-style`.
